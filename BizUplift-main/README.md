## 📌 Overview

BizUplift is a **production-ready festival marketplace** that connects local artisans with customers across India.

### It delivers:
- 🎨 Dynamic festival-based UI  
- 🤖 AI-powered price negotiation  
- 🛒 Complete e-commerce workflow  
- 👥 Community-driven engagement  

---

## ✨ Features

### 🎨 Festival Themes
- 10 festival themes (Diwali, Holi, Eid, etc.)
- Real-time UI updates using CSS variables
- Canvas-based particle animations

### 🤖 AI Price Negotiation
- Hinglish conversational AI (Claude API)
- Smart deal detection
- Negotiated price applied to cart

### 🛒 Shopping Flow
- Browse → Filter → Negotiate → Cart → Checkout → Track
- Coupons + credit points support
- Payments: UPI / Card / COD

### ⭐ Credit System
- Earn via purchases, reviews, referrals
- Redeem at checkout

### 🎁 Hamper Builder
- Budget-based gift creation
- Personalized hampers

### 👥 Community
- Reviews and seller stories
- Pre-seeded content
